# 函数建模&&函数与外部查询交互操作详解
## 1.Moduling Function
```python
FUNCTION(name, min_param, max_param, param_types[], return_type, func_type)

example:Function('CONCAT',1,3,['String','String','String'],'String','scalar')
```

| 参数 | 说明 |
|--------|----------------|
|name       |函数名称|
|min_type   |最小参数个数
|max_type   |最大参数个数|
|param_types[]|参数类型列表|
|return_type    |返回类型|
|func_type  |函数类型|

## 2.Interact with Query
### Example
```SQL
SELECT CONCAT('param(1)','param(2)','param(3)') FROM t1;
```
### 步骤
1. 随机选择***FUNCTION***:eg:CONCAT
    - 位于`select list`,则`func_type`=`random.choose`(`scalar`,`aggregate`,`window`)
    - 位于`Having`,则`func_type`(`aggregate`)
2. 逐个参数填入：
    - `expected_type` = `param_types[i]`:  String
    - 从t1中查找 `return_type`=`expected_type` 的 `expected_column` --> `param(i)`
    - if `expected_column` can't be find: `expected_Literal` --> `param(i)`
